FudanNLP主要是为中文自然语言处理而开发的工具包，也包含为实现这些任务的机器学习算法和数据集。

演示地址: http://jkx.fudan.edu.cn/nlp/query

FudanNLP目前实现的内容如下：

中文处理工具
    中文分词
    词性标注
    实体名识别
    句法分析
    时间表达式识别
信息检索
    文 本分类
    新闻聚类
    Lucene中文分词
机 器学习
    Average Perceptron
    Passive-aggressive Algorithm
    K-means
    Exact Inference
